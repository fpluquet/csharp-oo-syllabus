# Syllabus Programmation Orientée Objet en C#

Ce syllabus fait suite au cours d'introduction à la programmation C# (Q1).

## Installation

```bash
npm install
```

## Développement

```bash
npm run docs:dev
```

## Build pour production

```bash
npm run docs:build
```

## Prévisualiser le build

```bash
npm run docs:preview
```
