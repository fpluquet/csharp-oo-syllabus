## 📥 Télécharger

Vous pouvez télécharger tous les fichiers du projet en tant que dossier compressé ou les copier individuellement.

## 📂 Structure des fichiers

```
etape1-encapsulation/
├── Echecs.csproj
├── Couleur.cs
├── TypePiece.cs
├── Piece.cs
├── Plateau.cs
└── Program.cs
```

## 🚀 Comment utiliser ?

1. **Créer un dossier** pour votre projet
2. **Copier les fichiers** dans ce dossier
3. **Ouvrir un terminal** dans le dossier
4. **Exécuter** : `dotnet run`

## 💾 Fichiers sources

Les fichiers sont situés dans le dossier statique `/public/codes/etape1-encapsulation/`

### Fichiers à copier :

- `Echecs.csproj` - Configuration du projet
- `Couleur.cs` - Énumération des couleurs
- `TypePiece.cs` - Énumération des types de pièces
- `Piece.cs` - Classe Piece avec logique de déplacement
- `Plateau.cs` - Classe Plateau (8x8)
- `Program.cs` - Point d'entrée
